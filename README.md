# Equilibrium MLM Platform

Новый проект для управления MLM-структурой, разработанный с нуля. Система состоит из микросервисной архитектуры:

- **backend/** — Django REST API, отвечающий за пользователей, тарифы, структуру и бонусы.
- **frontend/** — SPA-интерфейс администратора (React + TypeScript) для отображения дерева партнёров и работы с очередью.
- **infrastructure/** — конфигурация Docker/Compose, CI/CD и описание развёртывания.

## Основные возможности (MVP)

1. Регистрация пользователей и ведение очереди участников.
2. Завершение регистрации (оплата тарифа) с начислением двух бонусов — зелёного (вывод) и жёлтого (переход).
3. Отрисовка дерева партнёров с поддержкой zoom/pan, прогресса по уровням и отображением бонусов.
4. Очередь ожидающих участников с принятием оплаты и добавлением в структуру.
5. Health-check и интеграции (Telegram мини-приложение — в roadmap).

## Структура

```
backend/
  manage.py
  core/
  accounts/
  mlm/
  bonuses/
  api/
frontend/
  (будет настроен Vite/React)
infrastructure/
  docker-compose.yml
  Dockerfile.backend
  Dockerfile.frontend
```

## Развёртывание (в процессе)

- Создайте `.env` по примеру из `infrastructure/.env.example`.
- Запустите `docker-compose up --build` для локального окружения.
- В README будут дополняться инструкции по CI/CD и интеграциям.
